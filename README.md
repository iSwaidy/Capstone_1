# Capstone 1: EmporiUm Sales Territory Analysis

## Project Overview
Sales territory analysis for EmporiUm, a student bookstore chain selling tech products, 
art supplies, and more. This project analyzes Maryland in-store sales performance using 
SQL and Excel as part of the Year Up United Data Analytics Boot Camp.
